# Representation Learning of Protoplanetary Disks

This project studies morphological patterns in protoplanetary disks
using convolutional autoencoders and unsupervised clustering.

Pipeline

FITS data → dataset loader → autoencoder → latent vectors
→ clustering → morphology analysis

Methods
- Convolutional Autoencoder
- UMAP dimensionality reduction
- HDBSCAN clustering
- Radial brightness profile analysis
